# Unemployment Analysis with Python

## Objective
The objective of this project is to analyze unemployment trends in India using Python and visualize the impact of COVID-19 on unemployment rates across different regions.

## Dataset
- **Dataset Name:** Unemployment in India
- **File Used:** `unemp_upto_11_2020.csv`

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Project Workflow
1. Load the unemployment dataset.
2. Clean and preprocess the data.
3. Analyze unemployment rates across different regions.
4. Visualize monthly unemployment trends.
5. Compare unemployment rates of selected states.
6. Identify the top 10 states with the highest unemployment.
7. Perform correlation analysis.
8. Compare unemployment rates before and after the COVID-19 outbreak.

## Visualizations
- Average Unemployment Rate by Region
- Month-wise Unemployment Trend
- State-wise Unemployment Comparison
- Top 10 States with Highest Unemployment
- Correlation Heatmap
- Pre-COVID vs Post-COVID Unemployment Comparison

## Conclusion
The analysis shows that unemployment rates increased significantly during the COVID-19 pandemic. Regional variations are clearly visible, and the visualizations help understand the impact of the pandemic on employment across India.
