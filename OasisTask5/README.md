# Sales Prediction using Python

## Objective
The objective of this project is to predict product sales based on advertising expenditures across different media channels using Machine Learning.

## Dataset
- **Dataset Name:** Advertising Dataset
- **File Used:** `Advertising.csv`

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models
- Linear Regression
- Random Forest Regressor

## Project Workflow
1. Load and preprocess the dataset.
2. Remove unnecessary columns.
3. Perform Exploratory Data Analysis (EDA).
4. Visualize the relationship between advertising channels and sales.
5. Generate a correlation heatmap.
6. Split the dataset into training and testing sets.
7. Train Linear Regression and Random Forest Regression models.
8. Evaluate the models using MAE, RMSE, and R² Score.
9. Compare model performance.
10. Visualize feature importance and residuals.

## Visualizations
- Pair Plot
- TV vs Sales
- Radio vs Sales
- Newspaper vs Sales
- Correlation Heatmap
- Residual Plot
- Feature Importance Chart

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## Conclusion
The project demonstrates that television advertising has the strongest influence on product sales. Machine learning models successfully predict sales based on advertising budgets, with Random Forest generally providing better performance on this dataset.
