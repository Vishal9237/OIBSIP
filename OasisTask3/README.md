# Car Price Prediction with Machine Learning

## Objective
The objective of this project is to build a machine learning model that predicts the selling price of used cars based on various features such as fuel type, transmission, owner, brand, car age, and kilometers driven.

## Dataset
- **Dataset Name:** Car Details from Car Dekho
- **File Used:** `CAR DETAILS FROM CAR DEKHO.csv`

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models
- Linear Regression
- Random Forest Regressor

## Project Workflow
1. Load and preprocess the dataset.
2. Remove duplicate records.
3. Create new features such as car age and brand.
4. Perform Exploratory Data Analysis (EDA).
5. Visualize price distribution, fuel type comparison, car age analysis, and feature correlations.
6. Encode categorical variables.
7. Split the dataset into training and testing sets.
8. Train Linear Regression and Random Forest Regression models.
9. Compare model performance using MAE, RMSE, and R² Score.
10. Display feature importance for the Random Forest model.

## Visualizations
- Selling Price Distribution
- Selling Price by Fuel Type
- Car Age vs Selling Price
- Correlation Heatmap
- Feature Importance Chart

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## Conclusion
The Random Forest Regressor generally provides better prediction accuracy than Linear Regression because it can capture complex relationships between vehicle features and selling price. The project demonstrates the complete workflow of a machine learning regression problem, from data preprocessing and visualization to model evaluation.
