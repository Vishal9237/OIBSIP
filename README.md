# Oasis Infobyte Data Science Internship (OIBSIP)

## Internship Details
This repository contains the tasks completed as part of the Oasis Infobyte Data Science Internship Program (OIBSIP).

## Tasks Completed

### Task 1 - Iris Flower Classification
- Built machine learning models to classify Iris flowers.
- Compared Decision Tree, KNN, and SVM classifiers.

### Task 2 - Unemployment Analysis with Python
- Analyzed unemployment trends across India.
- Studied the impact of COVID-19 using data visualization.

### Task 3 - Car Price Prediction
- Predicted used car prices using machine learning.
- Compared Linear Regression and Random Forest models.

### Task 4 - Email Spam Detection
- Built a spam classifier using NLP.
- Compared Naive Bayes and Logistic Regression.

### Task 5 - Sales Prediction using Python
- Predicted product sales using advertising data.
- Evaluated Linear Regression and Random Forest models.
